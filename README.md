SkilloCity – A Learning Platform
A full-stack learning platform where students can connect with tutors, book sessions, chat in real-time, and join live classes.

🌐 Live Demo
Frontend (Vercel):
  https://skill-city-a-learning-platform.vercel.app
Backend (Render):
  https://skillocity-a-learning-platform.onrender.com

✨ Features
  🔐 User Authentication (JWT + Google OAuth)
  👨‍🏫 Tutor Discovery & Profiles
  📅 Session Booking System
  💬 Real-time Chat (Socket.io)
  📎 File & Image Sharing
  🎥 Google Meet Integration (Join Room)
  ⭐ Reviews & Ratings
  🔔 Notifications & Alerts

🏗️ Tech Stack
Frontend
  React (Vite)
  Tailwind CSS
  Axios
  Socket.io Client

Backend
  Node.js
  Express.js
  MongoDB (Atlas)
  Socket.io
  JWT Authentication
  Passport.js (Google OAuth)

Cloud / Services
  MongoDB Atlas
  Cloudinary (file uploads)
  Render (backend hosting)
  Vercel (frontend hosting)

1️⃣ Clone Repo
  git clone https://github.com/your-username/SkilloCity_A_Learning_Platform.git
  cd SkilloCity

2️⃣ Backend Setup
  cd backend
  npm install
  npm run dev

3️⃣ Frontend Setup
  cd frontend
  npm install
  npm run dev

🌍 Deployment
Backend (Render)
  Root Directory: SkilloCity/backend
  Build: npm install
  Start: npm start
  
Frontend (Vercel)
  Root Directory: SkilloCity/frontend
  Add ENV variables
Deploy 🚀
